# 🖼️ Capturas de tela

Coloque aqui as imagens usadas na seção **Capturas de tela** do [README](../../README.md).

## Arquivos esperados

| Arquivo | Conteúdo sugerido |
|---------|-------------------|
| `dashboard.png` | Painel principal (Home) |
| `risk-map.png` | Mapa de risco interativo |
| `clipping.png` | Clipping diário com resumo por IA |
| `programs.png` | Programas estratégicos |

## Como gerar

1. Rode o projeto localmente: `npm run dev`
2. Abra <http://localhost:5173> no navegador
3. Capture a tela (recomendado: 1280×800 ou maior, formato PNG)
4. Salve os arquivos nesta pasta com os nomes acima

> 💡 Dica: mantenha as imagens abaixo de ~500 KB cada (use compressão PNG) para não pesar o repositório.
> Enquanto as capturas reais não existirem, os links do README aparecerão como imagem quebrada — isso
> é esperado e não afeta o funcionamento da aplicação.
